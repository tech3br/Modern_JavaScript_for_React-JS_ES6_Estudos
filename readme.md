# ESTUDOS DO CURSO REALIZADO SOBRE ES6 E SUAS FUNCIONALIDADES

## ES6 FEATURES - NOVAS FUNCIONALIDADES NO ES6

- Criação de variáveis usando `let` e `const`;
- Template Strings;
- Arrow Functions;
- Rest e Spread Operator;
- Desestruturação;
- Array functions: find() and findIndex();
- Classes;
- Promisses (Promessas);
- etc;
